# NodeWebAppGenerator
Generatore web app con Node.JS, che crea tutte le cartelle e i file principali che servono.
Prima di tutto esegui questo comando

``` bash
chmod a+x *.sh
```

## Installazione

``` bash
./install.sh
```

## Disinstallazione

``` bash
./uninstall.sh
```

## Reinstallazione

``` bash
./reinstall.sh
